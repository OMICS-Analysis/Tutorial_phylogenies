Phylogeny training

Chapter 2 Packages studied R

  • ape: It provides a structure for storing a phylogenetic tree (phylo class), as well as basic manipulation and plotting functions. The phylo class Its used by most R packages which deal with phylogenies, so it is important to understand it.

A complete list of functions: https://cran.r-project.org/web/packages/ape/refman/ape.html#node.depth

https://eeob-macroevolution.github.io/Practicals/Intro_to_Phylo/intro_to_phylo.html

  • ggtree: An extension of ggplot to plot phylo trees and also the treedata objects of treeio (see below)

  • treeio: Contains functions to read and write multiple tree formats. We will study these functions:

    • full_join: joins metadata with a phylo object.
    • drop.tip: to eliminate one or many tree.tips
    • tree.subbset

  • phyloseq

  • phytools
